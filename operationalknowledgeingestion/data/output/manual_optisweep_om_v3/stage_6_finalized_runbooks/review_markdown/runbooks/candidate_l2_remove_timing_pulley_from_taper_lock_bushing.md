## Summary

Remove the timing pulley assembly from the shaft by locking out the tipper, gaining access to the motor area if needed, removing the taper-lock bushing set screws, using one set screw in the threaded release hole to free the pulley from the bushing, and sliding the assembly off the shaft.

## When To Use

Use when servicing or removing the timing pulley assembly and the documented source procedure calls for freeing the timing pulley from the taper-lock bushing and shaft using the set screw release method in section 7.3.19.1 Removal.

## Do Not Use For

* Do not use for operator-performed work.
* Do not use hammering or impact removal of the pulley.
* Do not use alternate removal methods not documented in the source when the pulley does not break free.

## Safety And Operational Notes

* LOTO the tipper before beginning the procedure.
* Use safety glasses and gloves.
* Do not use the round end of the hex wrench.
* Do not hit the pulley with a hammer because the source states this will damage the gearbox bearings.
* This procedure involves mechanical disassembly and is not operator-safe.

## Access Or Tools Needed

* LOTO access for the tipper
* Replacement pulley part
* Safety glasses
* Gloves
* Torque wrench
* 1/8-in. hex wrench
* 8-mm hex wrench
* 3/8-in. wrench or socket
* Access to the station side panel and motor area

## Procedure Steps

### Step 1 — Lock out and tag out the tipper

**Responsible role:** L2_support

**Instruction:**
LOTO the tipper using the referenced Operator Station Lockout/Tagout procedure before performing any timing pulley removal work.

**Expected result:**
The tipper is in a locked out and tagged out condition and safe for maintenance access.

**Stop or Escalate If:**

* LOTO cannot be completed.
* Safe maintenance condition cannot be established.

---

### Step 2 — Remove the station side panel if needed

**Responsible role:** L2_support

**Instruction:**
If necessary, use a 3/8-in. wrench or socket to remove the nine self-tapping screws securing the side panel, then remove the station side panel to access the motor.

**Expected result:**
The station side panel is removed when needed and the motor area is accessible.

**Screens / Images:**

![artifact_fig_7_20_timing_pulley](assets/artifact_fig_7_20_timing_pulley.png)

*Timing pulley assembly location and surrounding motor-area components used for access orientation.*


**Stop or Escalate If:**

* The side panel cannot be removed without damage.
* Required access to the motor area cannot be obtained.

---

### Step 3 — Remove both taper-lock bushing set screws

**Responsible role:** L2_support

**Instruction:**
Use a 1/8-in. hex wrench to remove both set screws from the bushing. Do not use the round end of the hex wrench.

**Expected result:**
Both set screws are removed from the taper-lock bushing without damaging the screw heads.

**Screens / Images:**

![artifact_fig_7_20_timing_pulley](assets/artifact_fig_7_20_timing_pulley.png)

*Set screw locations on the timing pulley and taper-lock bushing assembly.*


**Stop or Escalate If:**

* Either set screw cannot be removed.
* The hex interface is damaged or begins to strip.

---

### Step 4 — Identify the threaded release hole

**Responsible role:** L2_support

**Instruction:**
Identify the hole between the two set screw holes that has threads on the taper-lock bushing side and is smooth on the pulley side.

**Expected result:**
The correct center release hole is identified.

**Screens / Images:**

![artifact_fig_7_20_timing_pulley](assets/artifact_fig_7_20_timing_pulley.png)

*Relationship between the set screw holes, taper-lock bushing, and timing pulley to identify the center threaded release hole.*


**Stop or Escalate If:**

* The release hole cannot be positively identified.
* Hole geometry does not match the source description.

---

### Step 5 — Thread a removed set screw into the release hole

**Responsible role:** L2_support

**Instruction:**
Thread one removed set screw into the identified center hole.

**Expected result:**
One removed set screw is installed in the release hole and ready to be tightened.

**Screens / Images:**

![artifact_fig_7_20_timing_pulley](assets/artifact_fig_7_20_timing_pulley.png)

*Center release hole location relative to the two set screw holes.*


**Stop or Escalate If:**

* The set screw will not thread into the identified hole.
* Thread engagement is not possible or appears incorrect.

---

### Step 6 — Tighten the release screw to free the pulley

**Responsible role:** L2_support

**Instruction:**
Tighten the set screw in the center release hole to break the pulley free from the taper-lock bushing. The source notes this may take some work to get the pulley free.

**Expected result:**
The pulley breaks free from the taper-lock bushing.

**Screens / Images:**

![artifact_fig_7_20_timing_pulley](assets/artifact_fig_7_20_timing_pulley.png)

*Taper-lock bushing and timing pulley relationship while applying the release method.*


**Stop or Escalate If:**

* The pulley does not break free using the documented set screw method.
* The assembly shows signs of damage during release.

---

### Step 7 — Assist removal by hand only if needed

**Responsible role:** L2_support

**Instruction:**
If needed, push or pull the pulley by hand, but do not hit the pulley with a hammer because the source states this will damage the gearbox bearings.

**Expected result:**
The pulley is moved free without impact damage.

**Screens / Images:**

![artifact_fig_7_20_timing_pulley](assets/artifact_fig_7_20_timing_pulley.png)

*Pulley and gearbox relationship to avoid impact damage during final freeing.*


**Stop or Escalate If:**

* The pulley cannot be freed by hand after the documented release method.
* Any removal approach would require striking the pulley.

---

### Step 8 — Slide the pulley assembly off the shaft

**Responsible role:** L2_support

**Instruction:**
Once the pulley is free, slide everything off the shaft.

**Expected result:**
The timing pulley assembly is removed from the shaft.

**Screens / Images:**

![artifact_fig_7_20_timing_pulley](assets/artifact_fig_7_20_timing_pulley.png)

*Timing pulley assembly components and shaft relationship during final removal.*


**Stop or Escalate If:**

* The assembly will not slide off the shaft after being freed.
* Unexpected binding or obstruction remains.

---

## Success Criteria

* The timing pulley is freed from the taper-lock bushing using the documented set screw release method.
* The pulley assembly slides off the shaft.
* No hammering or impact damage occurs during removal.

## Failure Conditions

* LOTO cannot be established.
* The side panel cannot be removed when access is needed.
* The set screws cannot be removed.
* The correct release hole cannot be identified.
* The removed set screw cannot be threaded into the release hole.
* The pulley does not break free using the documented set screw method.
* The assembly does not slide off the shaft after being freed.
* Any attempt is made to strike the pulley with a hammer.

## Escalation Guidance

* Escalate if the pulley does not break free using the documented set screw method, because no additional removal method is provided in the source.
* Stop and escalate if removal would require striking the pulley.
* Escalate if the correct release hole cannot be identified from the source description and figure.
