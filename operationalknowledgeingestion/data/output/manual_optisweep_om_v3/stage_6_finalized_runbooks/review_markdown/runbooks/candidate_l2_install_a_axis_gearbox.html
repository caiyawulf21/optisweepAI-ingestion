Summary

Mount the A-axis gearbox in the documented orientation, secure it with eight M5x18 socket-head screws tightened to 5.1 Nm (45.1 in-lb) using blue or red Loctite®, reinstall referenced components, reinstall panels as needed, and restart the operator station using the referenced source procedures.

When To Use

Use this procedure when installing or re-installing the A-axis gearbox on the OptiSweep system after gearbox service or replacement, where the source directs the technician to mount the gearbox, orient the clamp collar access hole correctly, secure the gearbox fasteners, reinstall dependent components, and return the operator station to service.

Procedure Steps

Step 1 — Lift the A-axis gearbox into place

Responsible role: L2_support

Instruction:

Lift the A-axis gearbox into place at the mounting location.

Expected result:

The gearbox is positioned for installation.

Screens / Images:

Stop or Escalate If:

  • The gearbox cannot be safely lifted into place.
  • The gearbox cannot be positioned at the mounting location.

Step 2 — Orient the clamp collar access hole

Responsible role: L2_support

Instruction:

Orient the clamp collar access hole toward the rounded side of the carriage.

Expected result:

The clamp collar access hole faces the rounded side of the carriage.

Screens / Images:

Stop or Escalate If:

  • The gearbox cannot be oriented with the clamp collar access hole toward the rounded side of the carriage.

Step 3 — Install the gearbox mounting screws

Responsible role: L2_support

Instruction:

Install the eight (8) M5x18 socket-head screws.

Expected result:

All eight specified screws are installed in the gearbox mounting points.

Screens / Images:

Stop or Escalate If:

  • The specified hardware is unavailable.
  • The screws cannot be installed correctly.

Step 4 — Torque the mounting screws

Responsible role: L2_support

Instruction:

Tighten the eight screws to 5.1 Nm (45.1 in-lb).

Expected result:

All eight mounting screws are tightened to the specified torque.

Stop or Escalate If:

  • The required torque cannot be completed.

Step 5 — Apply Loctite® to the fasteners

Responsible role: L2_support

Instruction:

Use blue or red Loctite® on the fasteners.

Expected result:

The gearbox fasteners have blue or red Loctite® applied as specified.

Stop or Escalate If:

  • Required Loctite® application cannot be completed.

Step 6 — Install the motor using the referenced procedure

Responsible role: L2_support

Instruction:

Install the motor into the gearbox using the referenced "Motor Installation" procedure on page 108.

Expected result:

The motor is installed into the gearbox per the referenced procedure.

Screens / Images:

Stop or Escalate If:

  • The referenced Motor Installation procedure is not available.
  • Motor installation cannot be completed using the referenced procedure.

Step 7 — Install the gripper using the referenced procedure

Responsible role: L2_support

Instruction:

Install the gripper onto the system using the referenced "Installation" procedure on page 100.

Expected result:

The gripper is installed per the referenced procedure.

Stop or Escalate If:

  • The referenced gripper Installation procedure is not available.
  • Gripper installation cannot be completed using the referenced procedure.

Step 8 — Reinstall counter-balance weights if previously removed

Responsible role: L2_support

Instruction:

Install the counter-balance weights if they were previously removed, using the referenced installation procedure on page 96.

Expected result:

Counter-balance weights are reinstalled when applicable using the referenced procedure.

Screens / Images:

Stop or Escalate If:

  • The referenced counter-balance installation procedure is not available.
  • Counter-balance reinstallation cannot be completed using the referenced procedure.

Step 9 — Reinstall top and bottom side panels as needed

Responsible role: L2_support

Instruction:

Reinstall the top and/or bottom side panels if necessary.

Expected result:

Required side panels are back in place.

Stop or Escalate If:

  • Required panels cannot be reinstalled.

Step 10 — Restart the operator station using the referenced procedure

Responsible role: L2_support

Instruction:

Re-start the operator station using the referenced "Starting The Operator Station" procedure on page 66.

Expected result:

The operator station is restarted using the referenced procedure.

Stop or Escalate If:

  • The referenced Starting The Operator Station procedure is not available.
  • The operator station cannot be restarted using the referenced procedure.
