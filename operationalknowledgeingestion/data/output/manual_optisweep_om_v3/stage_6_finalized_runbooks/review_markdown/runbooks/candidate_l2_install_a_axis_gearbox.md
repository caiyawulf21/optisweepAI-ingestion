## Summary

Mount the A-axis gearbox in the documented orientation, secure it with eight M5x18 socket-head screws tightened to 5.1 Nm (45.1 in-lb) using blue or red Loctite®, reinstall referenced components, reinstall panels as needed, and restart the operator station using the referenced source procedures.

## When To Use

Use this procedure when installing or re-installing the A-axis gearbox on the OptiSweep system after gearbox service or replacement, where the source directs the technician to mount the gearbox, orient the clamp collar access hole correctly, secure the gearbox fasteners, reinstall dependent components, and return the operator station to service.

## Do Not Use For

* Do not use this runbook as a substitute for the referenced Motor Installation procedure on page 108.
* Do not use this runbook as a substitute for the referenced gripper Installation procedure on page 100.
* Do not use this runbook as a substitute for the referenced counter-balance installation procedure on page 96.
* Do not use this runbook as a substitute for the referenced Starting The Operator Station procedure on page 66.

## Safety And Operational Notes

* This is not marked support-safe in the candidate and involves mechanical installation and lifting.
* Use only the specified hardware and torque values supported by the source packet.
* This procedure depends on other referenced procedures for motor installation, gripper installation, counter-balance installation, and operator station restart details.

## Access Or Tools Needed

* Physical access to the A-axis gearbox mounting location
* Eight (8) M5x18 socket-head screws
* Torque tool capable of 5.1 Nm (45.1 in-lb)
* Blue or red Loctite®
* Access to referenced procedures on pages 96, 100, 108, and 66

## Success Criteria

* The A-axis gearbox is installed with the clamp collar access hole facing the rounded side of the carriage.
* Eight M5x18 socket-head screws are installed.
* The eight screws are tightened to 5.1 Nm (45.1 in-lb).
* Blue or red Loctite® is used on the fasteners.
* Referenced components are reinstalled as applicable.
* Top and/or bottom side panels are reinstalled as needed.
* The operator station is restarted using the referenced procedure.

## Failure Conditions

* The gearbox cannot be oriented with the clamp collar access hole toward the rounded side of the carriage.
* Specified hardware is unavailable.
* Required torque cannot be completed.
* Required Loctite® application cannot be completed.
* Referenced procedures needed for motor, gripper, counter-balance, or operator station restart are unavailable or cannot be completed.

## Escalation Guidance

* Escalate if the gearbox cannot be oriented with the clamp collar access hole toward the rounded side of the carriage.
* Escalate if the specified hardware is unavailable or if the required torque or Loctite® application cannot be completed.
* Escalate when completion depends on referenced procedures whose details are not present in this source packet.
